# Virtual Stock Market Simulation (C)

A menu-driven, console-based stock market simulation written in C.

## Features
- User signup and login
- Buy and sell stocks
- Portfolio and balance management
- File-based data persistence
- Real-time stock price visualization
- Historical stock trend display

## Files
- `demo.c` – main source code
- `users.txt`, `portfolio.txt` – generated at runtime
- `stock_trend_of_*.txt` – sample stock trend data files

## How to Run
```bash
gcc demo.c -o demo
./demo
